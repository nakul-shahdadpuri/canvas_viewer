


## p5-js-IDE
A p5.js cross-platform IDE written in electron.

## Setup
```sh
git clone https://github.com/nakul-shahdadpuri/p5-js-IDE.git
cd p5-js-IDE/
npm install
```

## To Run
```sh
npm start
```
